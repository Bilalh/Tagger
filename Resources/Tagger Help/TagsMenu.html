Editing Tools

Case Transformation

Case transformation are in the Tags menu. This allow changing the case of the fields of the selected files to either UPPER CASE, lower case or Title Case.

Tag Manipulation

  • Trim whitespace removes the whitespace from the beginning and end of file.
  • Renumber Files renumbers the select files starting from 1.
  • Swap First and Last Name swaps the first and last name, this works with multiple artists as well. e.g. if the artist was 'first1 last2, first2 last2, first3 last3' then it would be converted to 'last2 first1, last2 first2, last3 first3'

Filenames

The files can be renamed used one of the predefined formats or using a custom format, the extension is appended automatically.

The tags can also be obtained from filenames, either using one of the predefined formats or using a custom format. The tags other than those obtained from the filename are preserved.

Deleting Tags

Specific tags can from the files can be deleted using the Delete Tags submenu. It also allows deleting all tags.
