Basic Editing

Once file(s) are selected, they can be edited in the panel at the right hand side. If multiple files are selected, all of the files' tags are edited.

The files are immediately saved after editing is finished. If are in the middle of change the tags and want to discard them, use Undo( âŒ˜Z).

Multiple Files Editing

Multiple Files can edited at the same time. If any of the tags differ, placeholder values will be shown instead. Any changes will effect all of the selected files.

Artwork

Artwork can added by dragging the artwork to panel in the artwork tab. It can be deleted by using backspace or delete

Artwork can be saved to file by dragging the artwork to an folder or the desktop.
