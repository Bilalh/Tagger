Browsing

The left panel of the main window is the file browser. This is used browsing and for selecting the files to be edited. To change directory either use the file browser, the Go menu or the popup button.

Previous locations can be accessed using the navigation buttons Forward and Back. This can also be done using the menu items in the Go menu.

Columns

The columns that are displayed in the file browser can be changed by going to the Preferences. There is support for most column types.
